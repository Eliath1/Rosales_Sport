# RS mockup asset packs (configurator preview)

Place client-delivered template packs here. One folder per `templateSlug`.

Example:

```
classic-button/
  front-base.png
  back-base.png
  front-mask-body.png
  front-mask-sleeve.png
  front-mask-collar.png
  back-mask-body.png
  back-mask-sleeve.png
  front-shadow.png
  back-shadow.png
  placement.json
```

Requirements: [docs/specs/uniform-mockup-asset-brief-photographer.md](../../docs/specs/uniform-mockup-asset-brief-photographer.md)

Until packs arrive, `preview-compositor.js` uses placeholder photos from `demo/images/`.

Source files (Vecteezy demo): `source/vecteezy-48973378-preview.jpg`, `.png`, `.eps`
