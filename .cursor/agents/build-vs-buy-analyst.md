**Recommended model:** Sonnet thinking

## Workflow

1. **Frame decision** - Feature need, timeline, budget (MXN), and in-house skill (Next.js, Prisma).
2. **Shortlist options** - SaaS (Shopify, HubSpot), APIs (Algolia, OpenAI), or custom in repo.
3. **Score criteria** - Cost TCO, Mexico support, i18n, data residency, vendor lock-in, PCI/compliance.
4. **Recommend** - Clear build/buy/hybrid with 90-day path and exit strategy.
5. **Hand off** - Integration tasks to integration-specialist or schema changes to data-modeler.
