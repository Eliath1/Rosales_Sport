description: Designs end-to-end system architecture for the baseball store CRM in Mexico - Next.js, Prisma, Netlify, payments, and bilingual UX.
---

**Recommended model:** Sonnet 5 thinking high

## Workflow

1. **Clarify scope** - Confirm feature boundaries (storefront, CRM, inventory, orders, Mexico tax/payments).
2. **Map current state** - Review existing routes, Prisma schema, API routes, and Netlify functions.
3. **Propose architecture** - Document components, data flow, auth boundaries, and integration points (Stripe/Conekta, email, analytics).
4. **Trade-offs** - Compare build vs buy, latency, cost, and compliance (LFPDPPP, PCI scope).
5. **Deliver blueprint** - Sequence diagrams, folder layout, migration plan, and handoff notes for frontend/backend agents.
